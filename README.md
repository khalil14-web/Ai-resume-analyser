## âœ¨ Introduction

AI Resume Analyzer is a project built with React, React Router, and Puter.js that allows users to upload and store resumes and match them with job listings using AI. It provides a clean, reusable UI with ATS evaluations and custom feedback for each job listing.

## âš™ï¸ Tech Stack

- **React**: JavaScript library for building user interfaces with reusable components.
- **React Router v7**: Routing library for React with nested routes, data loaders, and client-side navigation.
- **Puter.js**: Browser-based library for authentication, storage, database, and AI functionalities.
- **Tailwind CSS**: Utility-first CSS framework for fast UI development.
- **TypeScript**: Adds static typing to JavaScript for better tooling and error detection.
- **Vite**: Fast build tool and development server for modern web projects.
- **Zustand**: Minimal and efficient state management library for React.

## ğŸ”‹ Features

- **Browser-based Auth**: Manage authentication entirely in the browser.
- **Resume Upload & Storage**: Users can upload and securely store resumes.
- **AI Resume Matching**: Evaluate resumes against job listings and get ATS scores and feedback.
- **Reusable UI Components**: Clean, maintainable, and modern interface.
- **Cross-Device Compatibility**: Fully responsive design.
- **Code Reusability**: Modular components for efficient development.
